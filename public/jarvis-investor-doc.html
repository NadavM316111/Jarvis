Real Example: Connecting a TV

A user asked JARVIS to "play music on the living room TV." The TV wasn't in the system. Here's what happened:

Step 1: Discovery

JARVIS scanned the home network and found an unknown device at IP 192.168.4.25

Step 2: Identification

It probed the device, discovered it was a Technicolor Fision TV using the DIAL protocol

Step 3: Research

JARVIS searched the web to learn how DIAL protocol works

Step 4: Integration

It wrote new code to communicate with this TV type, tested it, and saved it for future use

Step 5: Memory

JARVIS remembered this device permanently - next time, it just works

No human programmer was involved. JARVIS taught itself.
