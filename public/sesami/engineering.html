Technical Specifications

The core technologies powering Sesami's intelligent monitoring

Hardware

  • Camera: 4K Ultra HD, 160deg FOV
  • Processor: Qualcomm QCS610
  • Memory: 4GB RAM, 32GB Storage
  • Connectivity: WiFi 6, BLE 5.0
  • Audio: 4-mic array + speaker

AI/ML Stack

  • Vision: TensorFlow Lite + MediaPipe
  • Pose: Custom BlazePose variant
  • Classification: MobileNetV3
  • Inference: Edge TPU accelerated
  • Cloud: GCP Vertex AI pipelines

Security

  • Encryption: AES-256 end-to-end
  • Auth: OAuth 2.0 + biometric
  • Storage: HIPAA-compliant cloud
  • Privacy: On-device processing first
  • Compliance: SOC 2 Type II

AI Milestone Detection

How we identify developmental progress through computer vision

Motor Milestones

Our pose estimation model tracks 33 body keypoints to detect motor development.

0-3mo

Head lifting, arm/leg movement patterns

3-6mo

Rolling over, reaching, grasping objects

6-9mo

Sitting, crawling initiation, standing support

9-12mo

Cruising, first steps, fine motor skills

Social-Emotional

Facial expression and interaction analysis for social development.

0-3mo

Social smiling, eye contact duration

3-6mo

Laughing, recognizing faces, responding to emotions

6-9mo

Stranger awareness, interactive play

9-12mo

Imitation, pointing, showing objects
