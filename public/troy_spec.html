Section 01

System Overview

The TroyAI Dynamic Market Intelligence Network is a live, interactive force-directed graph that maps the relationships between macro-economic forces, geopolitical actors, market sectors, and individual instruments. It is the visual and analytical backbone of TroyAI's recommendation engine — every buy/sell/hold signal traces back to a chain of connected nodes in this graph.

Unlike a static diagram, this network updates daily (or intraday) from live news feeds, economic data releases, and market price action. Node signal strengths pulse in real time. New events inject as temporary nodes and show their chain-reaction connections. Students and subscribers can overlay their own portfolios and see exactly how macro forces are hitting their specific holdings.

How It Integrates with TroyAI

The network graph is not cosmetic — it is the reasoning map that TroyAI uses internally. When TroyAI makes a recommendation, it:

  1. Reads the current state of all nodes (signal strength, recent events, price triggers)
  2. Traverses the link graph outward from the triggering node up to 3 hops
  3. Applies TROY investment logic to determine action: BUY / SELL / HOLD / WATCH
  4. Returns a structured recommendation with chain-reaction explanation
  5. Displays the reasoning visually in the network graph for the student

Data Model

Node Schema

Every entity in the network (macro force, geopolitical actor, sector, indicator, risk event) is a Node object.

TypeScriptinterface TroyNode {
  id:          string;         // Unique slug e.g. "fed", "china", "tech"
  label:       string;         // Display name on graph
  type:        NodeType;       // "macro" | "geo" | "sector" | "risk" | "data" | "portfolio"
  r:           number;         // Node radius (visual size = importance)
  signal:      number;         // 0–100 current signal strength (updates live)
  delta:       number;         // Signal change in last 24h (-100 to +100)
  safetyScore: number;         // 0–100 risk score (100 = safest)
  action:      string;         // TROY monitoring action label
  desc:        string;         // Chain-reaction description
  rec:         string;         // TROY recommendation string
  ticker?:     string;         // Optional: associated ticker symbol
  sentiment?:  number;         // -1 to +1 news sentiment score
  lastEvent?:  string;         // Description of most recent triggering event
  updatedAt:   Date;           // Last data refresh timestamp
  isLive:      boolean;        // true = actively updating from live feeds
  newsItems?:  NewsItem[];     // Recent news articles mapped to this node
}

type NodeType = "macro" | "geo" | "sector" | "risk" | "data" | "portfolio" | "event";

Link Schema

TypeScriptinterface TroyLink {
  id:          string;         // Auto-generated: "source_id-target_id"
  source:      string;         // Source node ID
  target:      string;         // Target node ID
  strength:    1 | 2 | 3;      // 1=weak, 2=moderate, 3=strong
  direction:   LinkDirection;  // "unidirectional" | "bidirectional"
  type:        LinkType;       // "causal" | "correlational" | "reactive"
  label?:      string;         // Optional: edge label e.g. "Controls", "Impacts"
  active:      boolean;        // false = link temporarily dormant (low signal)
  liveWeight?: number;         // Real-time computed weight (correlation rolling 30d)
}

type LinkDirection = "unidirectional" | "bidirectional";
type LinkType = "causal" | "correlational" | "reactive";

News Item Schema

TypeScriptinterface NewsItem {
  id:          string;
  headline:    string;
  source:      string;         // "Reuters" | "Bloomberg" | "CNBC" | "X" | etc.
  url:         string;
  publishedAt: Date;
  sentiment:   number;         // -1 (bearish) to +1 (bullish)
  nodeIds:     string[];       // Nodes this article affects e.g. ["fed","rates","tech"]

Node Click — Chain Reaction Logic

JavaScriptfunction onNodeClick(event, d) {
  // 1. Find all directly connected nodes (1 hop)
  const directConnections = new Set();
  links.forEach(l => {
    const s = l.source.id || l.source, t = l.target.id || l.target;
    if (s === d.id) directConnections.add(t);
    if (t === d.id) directConnections.add(s);
  });

  // 2. Find 2-hop connections
  const secondaryConnections = new Set();
  directConnections.forEach(connId => {
    links.forEach(l => {
      const s = l.source.id || l.source, t = l.target.id || l.target;
      if (s === connId && !directConnections.has(t) && t !== d.id)
        secondaryConnections.add(t);
    });
  });

  // 3. Highlight graph
  nodeSelection.select('circle')
    .attr('fill-opacity', n =>
      n.id === d.id ? 0.6 :
      directConnections.has(n.id) ? 0.35 :
      secondaryConnections.has(n.id) ? 0.2 : 0.05
    );

  // 4. Fetch TROY analysis from backend
  fetchChainReactionAnalysis(d.id).then(analysis => {
    renderInfoPanel(d, analysis);
  });
}

async function fetchChainReactionAnalysis(nodeId) {
  const res = await fetch(`/api/analysis/chain-reaction?node=${nodeId}`);
  return res.json();
}

Node Signal Score Algorithm

JavaScriptfunction calculateSignalScore(node, recentNewsItems, priceData) {
  const weights = {
    newsVolume:  0.30,  // Number of articles in last 24h
    sentiment:   0.25,  // Average sentiment of recent articles
    priceAction: 0.25,  // Price momentum if node has ticker
    baseSignal:  0.20,  // Node's intrinsic importance
  };

  const newsVolScore = Math.min(recentNewsItems.length / 20, 1) * 100;
  const sentimentScore = ((recentNewsItems.reduce((a, b) => a + b.sentiment, 0) 
    / recentNewsItems.length) + 1) * 50;
  const priceScore = priceData ? momentumScore(priceData) : 50;

  return (
    newsVolScore * weights.newsVolume +
    sentimentScore * weights.sentiment +
    priceScore * weights.priceAction +
    node.baseSignal * weights.baseSignal
  );
}

Visual Encoding

JavaScript// Add safety score ring to each node
nodeG.append('circle')
  .attr('class', 'safety-ring')
  .attr('r', d => d.r + 4)
  .attr('fill', 'none')
  .attr('stroke', d => safetyColor(d.safetyScore))
  .attr('stroke-width', 2)
  .attr('stroke-dasharray', d => {
    // Score as arc: full circle = 100, partial = score%
    const circumference = 2 * Math.PI * (d.r + 4);
    const filled = (d.safetyScore / 100) * circumference;
    return `${filled} ${circumference - filled}`;
  })
  .attr('stroke-dashoffset', d => {
    const circumference = 2 * Math.PI * (d.r + 4);
    return circumference / 4; // Start arc from top
  });
